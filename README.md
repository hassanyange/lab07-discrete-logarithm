# Лабораторная работа №7: Дискретное логарифмирование в конечном поле

## Описание
Реализация алгоритма дискретного логарифмирования в конечном поле GF(p) с использованием ρ-метода Полларда.

## Теоретическая основа
**Задача дискретного логарифмирования**: Для заданных простого числа p и целых чисел a, b найти x такой, что:
\[
a^x \equiv b \ (\text{mod} \ p)
\]

**ρ-метод Полларда**:
- Использует идею "черепахи и зайца" для поиска коллизий
- Применяет ветвящееся отображение
- Основан на вероятностном алгоритме
